# FKLineChart

## Installation

### CocoaPods

Add the following entry to your Podfile:

```rb
pod 'FKLineChart'
```

Then run:

```rb
pod install
```

Don't forget to import FKLineChart in every file you'd like to use line chart :

```swift
import FKLineChart
```

## Usage

1- Add a UIView to your storyboard and set its custom class `FKLineChart`.
